# N・S高プロフィール画像作っちゃお

## Overview
N・S高校の生徒がTwitterなどで自己紹介する際に使用できるプロフィール画像を生成することができるサービスです。<br>
- サービスのURLはこちら→[N・S高プロフィール画像作っちゃお](https://nsprofile-generator.netlify.app)
- N・S高について知りたい方はこちら→[N・S高校](https://nnn.ed.jp/)

## Requirement
- jQuery 3.2.1
